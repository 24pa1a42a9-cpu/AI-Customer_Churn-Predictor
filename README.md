📊 AI Customer Churn Predictor

An end-to-end Machine Learning web application that predicts whether a telecom customer is likely to churn based on customer behavior and subscription details.

---

🚀 Project Overview

Customer churn prediction is an important business problem in the telecom industry. This project uses Machine Learning techniques to analyze customer data and predict the likelihood of customer churn.

The project includes:

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning Model Training
- Streamlit Web Application
- Cloud Deployment

---

✨ Features

- Predicts customer churn in real-time
- Interactive Streamlit user interface
- Multiple customer input features
- Machine Learning classification model
- Online deployed application

---

🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

---

📂 Dataset

The project uses the Telecom Customer Churn dataset containing customer subscription details, billing information, and service usage patterns.

---

⚙️ Machine Learning Workflow

1. Data Understanding
2. Data Cleaning
3. Handling Missing Values
4. Encoding Categorical Features
5. Feature Selection
6. Model Training
7. Model Evaluation
8. Deployment using Streamlit

---

📈 Model Performance

- Model Used: Random Forest Classifier
- Accuracy Achieved: 75%
