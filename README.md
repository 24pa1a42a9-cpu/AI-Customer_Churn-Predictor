# AI Customer Churn Predictor

An end-to-end Machine Learning web app that predicts whether a telecom customer is likely to churn.

## Features
- Machine Learning prediction
- Streamlit web app
- Interactive UI
- Real-time predictions

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

75%
